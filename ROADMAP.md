# Roadmap

## Phase 1: planning and audit

Goal:

- establish the mission schema
- compile risk tiers and review gates
- generate run archives and dossiers

Deliverables:

- mission compiler CLI
- artifact archive format
- report generator

## Phase 2: scientific adapters

Goal:

- connect the planning layer to real model and simulator adapters

Deliverables:

- Evo 2 adapter for scoring, embedding, and proposal generation
- protein triage adapter for structure and interaction follow-up
- habitat twin runner for shock matrices
- BioControl DSL template compiler

## Phase 3: orchestration

Goal:

- turn the static plan into an executable workflow system

Deliverables:

- NexusFlow pipeline export
- OmniForge job orchestration
- local copilot and retrieval integration
- persistent review queue and alerting

## Phase 4: operator tooling

Goal:

- expose the stack through a mission studio and inspection dashboard

Deliverables:

- browser-based mission builder
- DNA and protein inspection panels
- controller observability views
- run comparison and artifact browser

## Phase 5: frontier innovation modules

Goal:

- extend the compiler with research-backed 2025-2026 innovations that push NexusBioTwin beyond standard digital twin stacks

Deliverables:

- Multi-Model Ensemble Genomic Engine (Evo 2 + GENERator + AlphaGenome) with calibrated uncertainty and inter-model disagreement detection
- Epigenomic Landscape Analyzer for CRE perturbation planning using EpiAgent-style chromatin encoding
- Single-Cell Atlas Mapping Tool for CellFM-style zero-shot transfer and retrieval
- Metabolic Flux Sandbox for cofactor, ATP, and pathway-economics review
- Cell-Free Rapid Prototyping Planner that gates organism-level engineering behind fast cell-free validation
- Active-Learning Cell-Free Biosensor Optimization tool for selectivity-aware assay search
- Autonomous Protein Evolution Biofoundry planner for round-gated DBTL cycles
- Biomineral Interface Forge for biofilm-material and shielding/filter co-design
- Bayesian Optimization Closed-Loop Experiment Designer with cognitive digital twin integration
- Chronobiology Pulse Scheduler for cycle-aware actuation under offworld or industrial rhythms
- Containment Circuit Lattice planner for layered auxotrophy, tripwire, and fail-closed logic
- Sim-to-Real Transfer Bridge with domain randomization specs and iterative Bayesian calibration
- CellSAM-powered imaging segmentation and morphology QC for assay and spatial evidence
- DNA Lineage Memory Recorder for event logging and post-run forensic replay
- Mars Regolith Bioremediation example mission exercising all innovation modules
- Expanded research reference index with 12 new anchoring papers
